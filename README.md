Coderwall-Node
==========

 A simple but extended Coderwall.com API Wrapper for Node.JS

##Current Features:
 - User information including about-me, number of total protips and profile avatar`getUserProfile()`
 - User information including about-me, number of total protips and profile avatar  `getUserProfile()`
 - User Latest 4 Protips `getUserLatestProtips()`
